🔐 User Authentication System
A full-featured web application that combines secure user authentication with an integrated Code Examiner and multiple online exam modules — including Python, Java, and Aptitude tests.

🚀 Features
🔑 Authentication

Sign Up — New user registration with form validation
Login — Secure user login with session management
Logout — Safe session termination and redirect

🏠 Home Dashboard

Personalized home page after login
Navigation to all available exam modules
User session display

💻 Code Examiner

Allows users to write, examine, and evaluate code
Supports code review and analysis functionality
